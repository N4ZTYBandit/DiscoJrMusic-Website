Commands

Default prefix: !

!play <url>

Plays a song from a YouTube link and adds it to the queue.

!skip

Skips the current track and moves to the next one in the queue.

!loop

Toggles loop mode on the current song.

!shuffle

Randomly shuffles the current queue.

!volume <1-100>

Sets the playback volume.

!stop

Stops the music player.
